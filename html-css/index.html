見出しとは？

見出しとは、文章の大まかな内容を示すものです。
文章を読む際、見出しはとても大切な要素です。
見出しさえ見れば、その文章がどういう内容であるかがわかるようなものが理想です。

見出しは順序を揃えて

見出しに使う HTML のタグは h1, h2, h3, h4, h5, h6 ですが、それらには順序があります。
h2 は h1 の子要素として使われます。数字の小さいものほど重要度も高くなります。
